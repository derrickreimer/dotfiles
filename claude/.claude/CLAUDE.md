# Global instructions

## Attribution

NEVER add Claude/AI attribution to any content I author or publish. This includes
but is not limited to:

- Commit messages
- Pull request titles and bodies
- Issue/ticket comments
- Code comments

Specifically, never append lines like "🤖 Generated with Claude Code",
"Co-Authored-By: Claude", or any equivalent attribution/credit to a tool or model.
Write everything as if I authored it directly.
